# Festas de São Pedro

🌐 **Live Website**: https://festasspedrocaide.netlify.app/

## Project Overview
The Festas de São Pedro website project is a dynamic and engaging platform developed using Astro and Tailwind CSS. It aims to provide information and resources about the Festas de São Pedro celebrations, showcasing local traditions and activities.

## Technologies Used
- **Astro**: A modern static site generator that allows developers to create fast websites using components from various frameworks.
- **Tailwind CSS**: A utility-first CSS framework that provides a comprehensive set of classes for creating custom designs without leaving your HTML.

## Features
- **Responsive Design**: The website is fully responsive, ensuring a seamless experience across devices of all sizes.
- **Dynamic Content**: Leveraging Astro's capabilities, content can be updated without needing a complete rebuild of the website.
- **Customization Options**: Tailwind CSS allows extensive customization of styling to align with the branding of the Festas de São Pedro.

## Installation
To run the project locally, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/DiogoMotaMoreira/FestasS-oPedro.git
   cd FestasS-oPedro
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm start
   ```
